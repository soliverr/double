# Определить операционную систему устройства

Необходимо определить операционную систему устройства, на котором производится установка Node.js. Необходимо определить класс операционной системы, например, Windows, Linux, Android и т.д., а также, по возможности, точную версию (дистрибутив) операционной системы.

## Требования

### Формат вывода

Формат вывода информации может быть различным и задаваться агентом или пользователем. Должны быть поддержаны следующие форматы вывода:
* text
* json
* yaml

Пример вывода в формате text:

```
Distributor ID:	Ubuntu
Description:	Ubuntu 22.04.5 LTS
Release:	22.04
Codename:	jammy
Kernel:     6.8.0-101-generic
Architecture: x86_64
```

### Входные параметры

При реализации данного умения, агент должен задачть в спецификации навыка, требование к формату вывода информации.

## Реализация

Реализация данного умения реализована в виде навыка [detect-operation-system](../skills/detect-operation-system/detect-operation-system.md).
