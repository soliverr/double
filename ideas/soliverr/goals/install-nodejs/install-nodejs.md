# Установка Node.js



## Цель

Установить Node.js для различных операционных систем. Модет существовать несколько вариантов установки Node.js для одной операционной системы. Пользователь должен выбрать предпочитаемый вариант установки для своей операционной системы. Агент Double должен помочь пользователю с выбором, предоставив рекомендации, а ткаже показать наличие установленной версии Node.js, способ, которым она была установлена, доступные для обновления версии. 

## Параметры валидации

- Node.js установлен
- Node.js обновлён до последней или указанной пользователем версии
- Node.js работает

# План

## Определить операционную систему устройства

Необходимо определить операционную систему устройства, на котором производится установка Node.js. Необходимо определить класс операционной системы, например, Windows, Linux, Android и т.д., а также, по возможности, точную версию (дистрибутив) операционной системы, например:

```
Distributor ID:	Ubuntu
Description:	Ubuntu 22.04.5 LTS
Release:	22.04
Codename:	jammy
```

## Определить способ установки Node.js

В зависимости от операционной системы, необходимо задать способ установки Node.js. Способ установки определяется пользователем, в зависимости от его предпочтений. А также в зависимости от архитектуры процессора на устройстве пользователя.


## Определить последнюю стабильную версию Node.js

Последнюю стабильную версию Obsidian необходимо определить на [официальной странице релизов проекта]( https://github.com/obsidianmd/obsidian-releases/releases)


## Определить наличие установленной версии Obsidian

В зависимости от операционной системы и способа установки, определить, установлен ли уже Obsidian на устройстве пользователя, и его версию.

Если установленная версия и последняя стабильная версия совпадают, то процесс установки/обновления завершён.

## Обновить Obsidian

Если установленная версия старее последней стабильной версии и пользователь хочет обновиться до последней стабильной версии, то обновить Obsidian до последней стабильной версии способом, в зависимости от операционной системы и выбранного пользователей способа установки Obsidian.

## Установить Obsidian

Если Obsidian не установлен, то установить выбранную пользователем версию (по умолчанию, последнюю стабильную версию) способом, в зависимости от операционной системы и выбранного пользователей способа установки Obsidian.

## Проверка установки

Проверить, что приложение Obsidian запускается, и его версия соответствует выбранной версии для установки/обновления.
