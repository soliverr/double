# Node.js

[Node.js](https://nodejs.org/en) - Node.js® is a free, open-source, cross-platform JavaScript runtime environment that lets developers create servers, web apps, command line tools and scripts.

Большинство агентов AI на данный момент реализованы на JavaScript или TypeScript, поэтому установка среды выполнения и разработки Node.js является одним из первых и существенных шагов для продолжения работы над агентами Double.
